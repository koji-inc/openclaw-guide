# OpenClaw完全攻略ガイド 🦞

> エンジニアのための OpenClaw 体系解説テキスト

## このガイドについて

OpenClawは、WhatsApp・Telegram・Discord・iMessageなど複数のチャットアプリをAIエージェントにつなぐ**セルフホスト型ゲートウェイ**です。

このガイドでは、OpenClawの仕組みをゼロから体系的に学べるカリキュラムを提供します。

## 対象読者

- プログラミング経験のあるエンジニア
- AIツールに興味がある人
- 自分専用のAIアシスタントを構築したい人
